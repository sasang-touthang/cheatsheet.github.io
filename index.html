HTML Tags Reference

Tag Name Description
<html> HTML The wrapper element for all HTML codes
<title> Title The wrapper element inside the HTML tag that displays the title of the web page
<link> Link The wrapper element that links one web page to another
<body> Body The wrapper element that displays the content of the web page
<h1> h1 The element inside the body tag that encloses the header text
<p> p The element use to write paragraph texts
